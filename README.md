## 🚀 部署详细教程

本项目是一个基于 React + Vite + Express 的全栈应用，支持多种部署方式。

### 方式一：Docker 部署 (推荐)

使用 Docker 可以最快速地在任何服务器上运行本项目，无需配置 Node.js 环境。

1. **克隆或下载项目代码到服务器**
   ```bash
   git clone https://github.com/dcxa521gi/baby-food-app
   cd baby-food-app
   ```

2. **构建 Docker 镜像**
   ```bash
   docker build -t baby-food-app .
   ```

3. **运行容器**
   将容器的 3000 端口映射到宿主机的 13521 端口：
   ```bash
   docker run -d -p 13521:3000 --name baby-food-app --restart always baby-food-app
   ```

4. **访问应用**
   在浏览器中访问 `http://你的服务器IP:13521` 即可使用。

### 方式二：宝塔面板 (BT Panel) Docker 部署 (推荐)

如果您使用宝塔面板，可以通过 Docker 管理器使用“命令”方式快速创建容器。

1. **进入宝塔 Docker 管理器**
   - 在宝塔面板左侧菜单点击 **Docker**。
   - 点击 **容器** 选项卡。

2. **使用“添加容器”或“命令行”创建**
   - 点击 **添加容器**，选择 **命令行** 模式。
   - 输入以下命令（请确保您已在服务器上执行了 `git clone` 并进入了项目目录）：
   ```bash
   # 构建镜像
   docker build -t baby-food-app .
   
   # 创建并运行容器
   docker run -d -p 13521:3000 --name baby-food-app --restart always baby-food-app
   ```

3. **访问应用**
   - 在宝塔的**安全**页面，放行 `13521` 端口。
   - 在浏览器中访问 `http://你的服务器IP:13521` 即可使用。

### 方式三：宝塔面板 (BT Panel) Node 项目部署

1. **安装 Node.js 环境**
   - 在宝塔面板的**软件商店**中，搜索并安装 **Node.js 版本管理器**。
   - 打开 Node.js 版本管理器，安装 **Node.js v16.20.2 或以上版本**。

2. **上传项目文件**
   - 在**文件**中，创建一个新目录（如 `/www/wwwroot/baby-food`）。
   - 将项目的所有文件上传并解压到该目录。
   - **注意**：由于 Node 16 不支持直接运行 TypeScript，请确保您上传的文件中包含 `dist` 和 `dist-server` 目录（在本地运行 `npm run build` 生成）。

3. **添加 Node 项目**
   - 在**网站** -> **Node项目** 中，点击**添加Node项目**。
   - **项目目录**：选择刚才上传的目录 `/www/wwwroot/baby-food`。
   - **启动选项**：选择“自定义启动命令”。
   - **启动命令**：输入 `npm run start`。
   - **项目端口**：填写 `3000`（注：您可以修改源码 `server.ts` 中的端口为 13521，或者在宝塔网站设置中添加反向代理将外部 13521 端口指向它）。
   - **运行用户**：选择 `www`。

4. **启动与访问**
   - 点击提交，宝塔会自动执行 `npm install` 安装依赖并启动项目。
   - 在宝塔的**安全**页面，放行 `13521` 端口。
   - 在浏览器中访问 `http://你的服务器IP:13521` 即可使用。

### 方式四：更新应用教程

当项目发布新版本时，请按照以下步骤更新：

#### Docker 环境更新：
```bash
cd baby-food-app
git pull
docker build -t baby-food-app .
docker stop baby-food-app
docker rm baby-food-app
docker run -d -p 13521:3000 --name baby-food-app --restart always baby-food-app
```

#### 宝塔 Node 环境更新：
1. 在宝塔文件管理器中进入项目目录。
2. 点击“终端”，执行 `git pull` 获取最新代码。
3. 执行 `npm install` 更新依赖。
4. 执行 `npm run build` 重新构建项目。
5. 在 Node 项目管理器中点击“重启”项目。
